На вкладке “Сервер” доступна вкладка “Создать сервер”.

На этой вкладке для создания сервера требуется указать: id оператора, инвентарный номер сервера, имя сервера, модель сервера и описание. После заполнение всех полей нажмите на кнопку “Создать сервер”.

Для создания нового пользователя во вкладке “Пользователь” нажмите на “Настройки”. Затем внизу нажмите на кнопку “Создать нового пользователя”.

В открывшейся вкладке заполните имя, логин, пароль, email и нажмите на кнопку “Подтвердить изменения ”.
